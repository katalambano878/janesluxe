# Jane's Luxe favicon pack

Generated from `public/logo.png` via `npm run generate:seo`.

| File | Use |
|------|-----|
| `favicon.ico` | Browser tab (all sizes) |
| `favicon-16x16.png` | Small tabs |
| `favicon-32x32.png` | Standard tabs |
| `favicon-48x48.png` | Windows tiles |
| `apple-touch-icon.png` | iOS home screen (180×180) |
| `android-chrome-192x192.png` | PWA / Android |
| `android-chrome-512x512.png` | PWA splash |

Copies are also placed at `public/favicon.ico`, `app/favicon.ico`, `app/icon.png`, and `app/apple-icon.png` for Next.js.

After updating the logo, run: `npm run generate:seo`
